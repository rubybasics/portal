Installing CouchDb
------------------

```sh
$ brew install couchdb               # install (I have 1.6.1)
$ couchdb -b 					               # start in bg
$ curl http://127.0.0.1:5984/        # test it
$ open http://localhost:5984/_utils/ # the web interface
```
